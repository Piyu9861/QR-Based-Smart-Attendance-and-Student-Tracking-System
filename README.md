# QR Based Smart Attendance and Student Tracking System

## Project Overview
This project is a web-based attendance and student tracking system developed using Python, Flask, and MySQL. It uses QR codes for attendance management.

## Features
- QR Code Attendance
- Student Tracking
- Admin Dashboard
- HOD Dashboard
- Attendance Reports

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Installation
1. Install Python
2. Install required packages:
   pip install -r requirements.txt
3. Import the database:
   qr_attendance_db_FINAL.sql
4. Run:
   python app.py
